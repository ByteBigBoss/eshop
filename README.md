# eShop - Ecommerce Web Application

## Java Institute Project

### Description
This project is a part of the Java Institute curriculum and aims to build a basic e-commerce application with comprehensive features, including both admin and web interfaces.

## Features
- **Product Management**: Efficiently manage your product catalog.
- **Shopping Cart**: Allow users to add and manage items in their cart.
- **User Authentication**: Implement secure user authentication mechanisms.
- **Admin Dashboard**: Provide a user-friendly interface for administrators to manage the platform.
- **Responsive Design**: Ensure a seamless experience across various devices.

## Technologies Used
- PHP
- MySQL
- HTML
- JavaScript
- Bootstrap

## API
- Payment Gateway

## Getting Started
1. Clone the repository.
2. Set up the required development environment.
3. Run the application.
